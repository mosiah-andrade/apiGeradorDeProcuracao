# ProcuraSolar - Landing Page

Uma landing page moderna para um SaaS de geração de procuração para energia solar. Sistema gratuito, instantâneo e seguro que suporta todas as 28 concessionárias de energia do Brasil.

##  Recursos Principais

-  **Instantâneo**: Gere procuração em segundos
-  **100% Gratuito**: Sem custos ocultos ou assinaturas  
-  **Seguro**: Criptografia de nível bancário
-  **Responsivo**: Funciona em todos os dispositivos
-  **Todas as Concessionárias**: Suporta 28 distribuidoras de energia
-  **Válido Legalmente**: Reconhecido por todas as concessionárias

##  Começar Rápido

### Pré-requisitos
- Node.js 18+
- npm/yarn/pnpm

### Instalação

`bash
npm install
`

### Desenvolvimento

`bash
npm run dev
`

Abra [http://localhost:3000](http://localhost:3000)
